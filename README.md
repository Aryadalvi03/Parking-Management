# 🚗 Smart Parking Management System

This is a modern web-based Smart Parking Management System developed using Next.js, TypeScript, and Tailwind CSS. The system allows users to search for available parking slots, register vehicles, make payments, and manage parking records efficiently.

The project is designed to provide a structured and user-friendly solution for managing vehicle entry, slot availability, and parking operations digitally.

## Features

- User login and registration system  
- Owner login functionality  
- Parking slot search system  
- Parking registration module  
- Online payment interface  
- Responsive and modern UI  
- Component-based architecture using TypeScript  

## Technologies Used

- Next.js  
- TypeScript  
- Tailwind CSS  
- HTML, CSS, JavaScript  
- PHP (for backend form handling)  

## Project Structure

The project follows a component-based architecture with reusable UI components and separate pages for different modules such as login, parking search, payment, and dashboard.

## How to Run the Project

Install the dependencies using npm, start the development server, and open the application in your browser to access the system.
